# Gachamii

A cute offline gachapon collecting game for Android. Drag coins, spin the machine, collect items. Built with Unity 6.2 (2D URP). Made for Nick.

## Features

- **Gachapon Machine** — Drag coins to the machine and watch balls bounce around before one pops out
- **Tap to Reveal** — Open gacha balls with a satisfying reveal animation
- **Collection System** — Create unique collectible items across categories and generations
- **Daily Rewards** — Earn coins just for logging in each day
- **Daily Codes** — Unique redeemable codes generated for every date
- **Advertisement** — Silly ad video with 3 day cooldown
- **Tutorial Scene** — Nick and Annie explain how the game works
- **Inventory Scene** — Scrollable inventory with the option to examine collectibles
- **Fully Offline** — All progress saved locally
